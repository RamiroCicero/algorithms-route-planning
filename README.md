# Highway Route Planning System

This project was developed as the final assignment for the *Algorithms and Data Structures* course at **Politecnico di Milano**.

The goal of the project is to implement a system that manages highway service stations and computes the optimal route between them using the minimum number of stops.

The project was evaluated with the highest grade: **30/30 cum laude (30 e lode)**.

## Problem Overview

Each service station is identified by its distance from the start of the highway and contains a set of electric vehicles with different driving ranges.

Given two stations, the system must compute the route requiring the minimum number of stops. At each stop, a new vehicle must be selected from the available ones.

The implementation focuses on algorithm efficiency and correct handling of dynamic operations such as:

- Adding and removing service stations
- Managing vehicles available at each station
- Computing the optimal route between two stations

## Implementation

- Language: **C (C11)**
- Input: **stdin**
- Output: **stdout**
- No external libraries
- No multithreading
